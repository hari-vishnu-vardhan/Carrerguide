Subjects & Concepts

  • DOM Manipulation (Selecting, Modifying, Adding Events)
  • Version Control (Git basics, Commit, Push, Pull)

Intermediate:

  • Responsive Web Design (Media queries, Mobile-first approach)
  • CSS Preprocessors (Sass, LESS)
  • JavaScript (ES6+, Arrow functions, Promises, Fetch API)
  • Frameworks (React.js or Angular - Components, State management)
  • JavaScript Libraries (jQuery, Lodash)

Advanced:

  • Advanced React.js (React Hooks, Context API, Redux)
  • Webpack, Babel (Module bundling, Transpiling)
  • CSS Frameworks (Bootstrap, Tailwind CSS)
  • Progressive Web Apps (Service workers, Manifest)
  • Cross-browser compatibility
  • Performance Optimization (Lazy loading, Minification, Tree-shaking)
